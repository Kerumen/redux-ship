Redux Ship

Scalable, testable and typable side effects for Redux

Redux Ship is a side effects handler for Redux which focuses on:

  • scalability: you can reuse and compose sub-stores, thanks to the Ship.map primitive;
  • testing: you can run unit tests of your side effects, by taking snapshots of their live execution traces;
  • typing: you can type check your code with (almost) full coverage in Flow.

Inspired by:

  • Elm for the composition mechanism and type checking;
  • Redux Saga for the use of generators to test side-effects;
  • Jest for the idea of snapshot testing.

See a demo with reusable components and shared state.

Install

Run:

npm install --save redux-ship

You can optionally install Flow to get type checking and redux-ship-logger to get logging.
